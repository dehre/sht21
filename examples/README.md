## Wiring up

Both examples use Pin 21 for SDA and Pin 22 for SCL:

```
   ______________                        ______________
  |              |                      |              |
  |       SDA(21)|______________________|SDA           |
  |              |                      |              |
  |       SCL(22)|______________________|SCL           |
  |              |                      |              |
  |           3V3|______________________|VCC           |
  |              |                      |              |
  |           GND|______________________|GND           |
  |              |                      |              |
  |_ESP32________|                      |_SHT21_Sensor_|

```
